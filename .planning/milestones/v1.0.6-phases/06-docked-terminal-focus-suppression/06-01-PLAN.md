   `Some(TerminalAction::Hovered) => { /* No-op: hover does not change focus */ }`

**bottom/mod.rs -- Float path (radky 66-86):**
4. Radek 18: Prejmenovat parametr `_dialog_open` na `dialog_open`
5. Radek 67: Pridat `&& !dialog_open` do is_focused:
   `let is_focused = ws_arg.focused_panel == FocusedPanel::Build && !dialog_open;`
   POZOR: `dialog_open` parametr je v outer funkci `render_bottom_panel`, ne v closure. Rust closures zachycuji promenne z enclosing scope. `dialog_open` je `bool` (Copy), takze borrow neni problem -- proste pouzit `dialog_open` uvnitr closure.

   - `TerminalAction::Clicked => { if !dialog_open { ws_arg.focused_panel = FocusedPanel::Build; } }`
   - `TerminalAction::Hovered => { /* No-op: hover does not change focus */ }`

**bottom/mod.rs -- Docked path (radky 152-161):**

   A pridat dialog guard na Clicked handler (radky 153-155):
