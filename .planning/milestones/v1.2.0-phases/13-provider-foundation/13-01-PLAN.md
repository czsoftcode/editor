2. Vytvorit `src/app/ai/provider.rs`:
   - `ProviderCapabilities` struct: `supports_streaming: bool`, `supports_tools: bool`
   - `ProviderConfig` struct: `base_url: String`, `model: String`, `temperature: f64`, `num_ctx: u64` — derive Clone, Debug, Serialize, Deserialize
   - `StreamEvent` enum: `Token(String)`, `Done { model: String, prompt_tokens: u64, completion_tokens: u64 }`, `Error(String)`, `ToolCall { id: String, name: String, arguments: serde_json::Value }`
   - `AiProvider` trait (Send + Sync): `name() -> &str`, `is_available() -> bool`, `available_models() -> Result<Vec<String>, String>`, `capabilities() -> ProviderCapabilities`, `config(&self) -> &ProviderConfig`, `send_chat(&self, messages: &[AiMessage], config: &ProviderConfig) -> Result<AiMessage, String>`, `stream_chat(&self, messages: Vec<AiMessage>, config: ProviderConfig) -> mpsc::Receiver<StreamEvent>`

3. Vytvorit `src/app/ai/ollama.rs`:
   - `OllamaStatus` enum: `Available(Vec<String>)`, `Unavailable`
   - `OllamaProvider` struct: `config: ProviderConfig`, `agent: ureq::Agent` (s timeout_read 300s)
   - `OllamaProvider::new(base_url: String, model: String)` — vytvori ureq Agent pres AgentBuilder s timeout_read(Duration::from_secs(300))
   - `impl AiProvider for OllamaProvider`:
     - `name()` vraci "ollama"
     - `is_available()` — GET {base_url}/api/tags, vraci true pri Ok response
     - `available_models()` — GET {base_url}/api/tags, parse JSON, vraci jmena modelu (stripnout ":latest" suffix, zobrazit jen nazev)
     - `capabilities()` — supports_streaming: true, supports_tools: false
     - `config()` — vraci &self.config
     - `send_chat()` — POST {base_url}/api/chat s stream:false, parse celou odpoved, vraci AiMessage
     - `stream_chat()` — spawne std::thread, POST {base_url}/api/chat s stream:true, cte pres BufReader::lines(), parsuje NDJSON radek po radku, posila StreamEvent pres mpsc::Sender. Pri tx.send() Err (receiver dropped) ciste ukonci thread.
   - Helper funkce (pub pro testovani): `parse_tags_response(json: &str) -> Result<Vec<String>, String>`, `parse_ndjson_line(line: &str) -> Option<StreamEvent>`
   - `spawn_ollama_check(base_url: String) -> mpsc::Receiver<OllamaStatus>` — spawne thread, GET /api/tags, posle OllamaStatus
