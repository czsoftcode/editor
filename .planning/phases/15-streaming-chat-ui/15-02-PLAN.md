**1. Stop/Send toggle v prompt area (render.rs, render_chat_content):**

Nahradit existujici Escape handler (radek ~217-219) kompletnim Stop tlacitkem:
- Pokud ws.ai.chat.loading == true:
  - Misto "Send" zobrazit cervene "Stop" tlacitko vedle prompt inputu (nebo v info baru). Barva tlacitka: `ui.visuals().error_fg_color` (semanticka barva pro destructive akci — NE hardcoded Color32)
  - Pri kliknuti na Stop NEBO Escape:
    1. ws.ai.cancellation_token.store(true, Relaxed)
    2. Zachovat castecnou odpoved: `if let Some(last) = ws.ai.chat.conversation.last_mut() { last.1 = format!("{}\n\n*[preruseno]*", ws.ai.chat.streaming_buffer); }`
    3. ws.ai.chat.streaming_buffer.clear()
    4. ws.ai.chat.loading = false
    5. ws.ai.chat.stream_rx = None (dropne Receiver, sender thread dostane SendError a ukonci se)
- Pokud ws.ai.chat.loading == false:
  - Zobrazit normalni Send tlacitko (nebo Enter shortcut jak je ted)

**2. Auto-scroll s moznosti zastaveni (render.rs, render_chat_content):**

Zmenit ScrollArea pro konverzacni historii:
- Pouzit `.stick_to_bottom(ws.ai.chat.auto_scroll)` misto `.stick_to_bottom(true)`
- Po show() zkontrolovat jestli uzivatel scrollnul nahoru:
  ```rust
  // Po scroll_out = ScrollArea::both()...show()

      let at_bottom = scroll_out.state.offset.y >= scroll_out.content_size.y - scroll_out.inner_rect.height() - 30.0;

      }

  ```
- Kdyz auto_scroll == false a loading == true, zobrazit "Scroll to bottom" tlacitko:
  ```rust
  if !ws.ai.chat.auto_scroll && ws.ai.chat.loading {
      ui.horizontal(|ui| {
          ui.with_layout(egui::Layout::right_to_left(egui::Align::Center), |ui| {
              if ui.small_button("v Scroll to bottom").clicked() {
                  ws.ai.chat.auto_scroll = true;
              }
          });
      });

  ```
