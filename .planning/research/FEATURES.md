# Milestone v1.3.0 Research - Features

## Table stakes
- AI terminal je otevritelny z UI a funguje chat send/receive.
- Streaming odpovedi funguje bez zamrzani UI.
- Model picker a slash/GSD prikazy zustavaji funkcni.
- Approval flow pro citlive operace zustava zachovan.

## Differentiators to preserve
- Jednoduchy local-first workflow bez externich cloud vazeb.
- Integrace s existujicim workspace stavem a terminal panely.

## Anti-features for this milestone
- Nove AI funkce a nove providery.
- Zmeny UX, ktere by menily ovladani panelu.
