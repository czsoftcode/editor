# Session notes — LSP implementace

## Co jsme udělali

### 1. LSP didOpen / didChange / didClose (`src/app/lsp/mod.rs`)
- Přidány 3 metody: `notify_did_open`, `notify_did_change`, `notify_did_close`
- Přidán `Arc<AtomicBool> initialized` — nastaví se na `true` až po dokončení handshaku
- Přidána metoda `is_initialized()` pro kontrolu před odesláním didOpen
- `workspace_folders` + `TextDocumentSyncClientCapabilities` v `InitializeParams`
  → bez toho rust-analyzer zpracovával soubory izolovaně (jen syntaktické chyby, ne typové)

### 2. Editor (`src/app/ui/editor/mod.rs`)
- `Tab` má nové pole `lsp_version: i32` (0 = didOpen ještě neodesláno)
- `ui()` přijímá `lsp_client: Option<&LspClient>` místo `diagnostics`
- `didOpen` se odesílá jen pokud `lsp.is_initialized()` — oprava race condition
- `didClose` se odesílá při zavření záložky pokud `lsp_version > 0`
- Helper `lang_id_from_path()` pro určení language ID

### 3. Render (`src/app/ui/editor/render.rs`)
- `ui_normal()` a `ui_markdown_split()` přijímají `lsp_client`
- `didChange` se odesílá jen pokud `lsp_version > 0` (didOpen byl odeslán)
  → oprava bugu: didChange před didOpen inkrementoval verzi → didOpen se nikdy neposlal

### 4. CPU optimalizace (`src/app/lsp/mod.rs`, `src/main.rs`)
- Tokio runtime: `new_multi_thread().worker_threads(2).enable_io()`
  → dříve `enable_all()` spouštěl N jader vláken (N = počet CPU jader)
- `request_repaint_after(100ms)` místo `request_repaint()`
  → rust-analyzer při indexování posílal desítky zpráv/s → editor renderoval 100+ fps

## Co jsme přidali (session 2)

### 5. Oprava zobrazování diagnostik (`render.rs`)
- Tečka v gutteru přesunuta doleva (left+6px), hover rect pokrývá celý gutter
- `paint_squiggles()` — 2px barevné podtrhání chybových řádků (červená/oranžová/modrá/zelená)
- Status bar: počet `✕ N  ⚠ N` — červeně/oranžově, jen pokud > 0

### 6. Hover dokumentace (`lsp/mod.rs`, `render.rs`)
- Mouse hover nad textem + 600ms debounce → `textDocument/hover` request
- Výsledek se zobrazí jako floating popup (500px max šířka, 240px max výška)
- Signature monospacově (modrá), popis separátorem

### 7. Go-to-definition (`lsp/mod.rs`, `render.rs`, `workspace/mod.rs`)
- **F12** na aktuální pozici kurzoru → `textDocument/definition` request
- Výsledek: otevře soubor + skočí na řádek
- Funguje pro Scalar/Array/Link odpovědi (vezme první)

### 8. Autocomplete (`lsp/mod.rs`, `render.rs`)
- **Ctrl+Space** → `textDocument/completion` request
- Floating dropdown: max 25 položek, šipky ↑↓ pro navigaci
- Enter/Tab = přijmout, Escape = zavřít, klik myší = přijmout
- Při přijmutí: nahradí aktuální slovo před kurzorem, přesune kurzor
