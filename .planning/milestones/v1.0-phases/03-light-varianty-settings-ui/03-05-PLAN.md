  <name>Task 2: Opravit teplý tón terminálu pro WarmIvory variantu</name>
  <files>src/app/ui/terminal/instance/theme.rs</files>
  <action>
Problém: `light_terminal_base_palette()` má `background: "#f3f5f7"` — studená neutrální šeď. `panel_fill` pro WarmIvory = (255, 252, 240). Blend `t=0.42` dává výsledek blízký bílé, ne teplý ivory tón.

Fix: Změň `background` v `light_terminal_base_palette()` z `"#f3f5f7"` na variantně-specifickou hodnotu. Protože `light_terminal_base_palette()` nezná variantu, nejjednodušší fix je přejmenovat funkci na `light_terminal_base_palette_for(panel_fill: egui::Color32)` a upravit background podle toho, zda je panel_fill teplý.

Konkrétní implementace:

1. Přidej pomocnou funkci pro detekci teplého tónu:

```rust
fn warm_ivory_bg(panel_fill: egui::Color32) -> &'static str {
    // WarmIvory panel_fill je (255,252,240) — r > b+20 signalizuje teplý tón
    if panel_fill.r() as i32 - panel_fill.b() as i32 > 10 {
        "#f5f2e8"  // teplejší základ pro ivory blending
    } else {
        "#f3f5f7"  // původní studená base pro CoolGray/Sepia
    }
}
```

2. V `light_terminal_base_palette()` změň `background` řádek na:

```rust
background: String::from("#f3f5f7"),
```

Ponech signaturu `light_terminal_base_palette()` beze změny — jsou na ní testy.

3. Místo toho uprav `tone_light_palette()` aby přijímal `panel_fill` explicitně a použil teplejší base background:

```rust
fn tone_light_palette(palette: ColorPalette, visuals: &egui::Visuals) -> ColorPalette {
    let tone = visuals.panel_fill;
    let bg_base = warm_ivory_bg(tone);
    ColorPalette {
        foreground: blend_hex(&palette.foreground, tone, 0.06),
        background: blend_hex(bg_base, tone, 0.55),  // teplejší base + vyšší ratio
        // ... zbytek beze změny
    }
}
```

Blend ratio background zvýš z 0.42 na 0.55 a přepni base z `&palette.background` na `bg_base`. Tím pro WarmIvory dostaneme blending `#f5f2e8` + teplá ivory barva t=0.55, výsledek ~(250,247,236) — viditelně teplý krémový tón.

Pro CoolGray a Sepia: `warm_ivory_bg` vrátí `"#f3f5f7"` (studená šeď), blend t=0.55 × neutrál = výsledek podobný původnímu (byl 0.42 × studená = ~248,248,244).

Ujisti se, že existující testy v `mod tests` stále projdou. Test `terminal_theme_light_background_is_light` požaduje luminance > 0.7 — (250,247,236) má luminance ~0.94, vyhovuje.
