# Requirements: PolyCredo Editor — Dark/Light Mode

**Defined:** 2026-03-04
**Core Value:** Uživatel přepne téma a celý editor odpovídá zvolenému režimu bez vizuálních defektů.

## v1 Requirements

### Theme Model

- [x] **THEME-01**: Settings obsahuje `LightVariant` enum (WarmIvory, CoolGray, Sepia) s `#[serde(default)]`
- [x] **THEME-02**: Settings obsahuje metody `to_egui_visuals()` a `syntect_theme_name()` odvozené z aktivního tématu
- [x] **THEME-03**: Téma je aplikováno při startu aplikace přes `cc.egui_ctx.set_visuals()` (bez úvodního bliknutí)
- [x] **THEME-04**: Změna tématu se propaguje přes existující `settings_version` mechanismus

### Editor — Syntax Highlighting

- [x] **EDIT-01**: Highlighter přijímá název syntect tématu jako parametr (bez hardcoded `"base16-ocean.dark"`)
- [x] **EDIT-02**: Light mode používá `"Solarized (light)"` syntect téma
- [x] **EDIT-03**: Dark mode používá `"base16-ocean.dark"` (stávající chování zachováno)
- [x] **EDIT-04**: Highlighter vyčistí cache pouze při skutečné změně tématu (ne každý frame)

### Terminál

- [x] **TERM-01**: Claude panel (egui_term) zobrazuje světlé pozadí v light mode
- [x] **TERM-02**: Build terminál (egui_term) zobrazuje světlé pozadí v light mode
- [x] **TERM-03**: Scrollbar terminálu není hardcoded tmavý — odvozuje se z aktivního tématu
- [x] **TERM-04**: Terminálový text (popředí) je čitelný na světlém pozadí (dostatečný kontrast)

### File Tree — Git Barvy

- [x] **TREE-01**: Git barvy (M/A/??/D) jsou čitelné na světlém pozadí (odlišné sady pro dark/light)
- [x] **TREE-02**: Untracked soubory (šedá) jsou viditelné v light mode (stávající `#3c3c3c` splývá)

### Light Mode Varianty

- [x] **LITE-01**: Teplá slonová kost — `panel_fill` přibližně `rgb(255, 252, 240)`, jemné teplé tóny
- [x] **LITE-02**: Studená šedá — `panel_fill` přibližně `rgb(242, 242, 242)`, GitHub/VS Code light styl
- [x] **LITE-03**: Béžová/sépiová — `panel_fill` přibližně `rgb(240, 230, 210)`, výraznější tónování
- [x] **LITE-04**: Každá varianta má správně nastaveno `faint_bg_color` pro viditelné odlišení panelů

### Settings Panel

- [x] **SETT-01**: Settings panel zobrazuje volič varianty light mode (3 možnosti), viditelný jen při light mode
- [x] **SETT-02**: Volba varianty se okamžitě projeví bez restartu (live preview)
- [x] **SETT-03**: Zvolené téma a varianta jsou persistovány v canonical `settings.toml` (`settings.json` slouží jen jako legacy vstup pro migraci)
- [x] **SETT-04**: Načtení starého `settings.json` (bez `light_variant`) neskončí pádem aplikace
- [x] **SETT-05**: Sandbox režim má v Settings snadno objevitelný tooltip a čitelnou poznámku o apply-on-reopen

### Sandbox Runtime Apply

- [x] **SANDBOX-01**: Změna sandbox režimu se po Save aplikuje okamžitě bez reopen a propaguje se do všech oken stejného projektu
- [x] **SANDBOX-02**: Save/Cancel semantika je zachovaná; OFF vyžaduje potvrzení; persist probíhá před runtime apply; při chybě persistu je nabídnuta volba revert/keep
- [x] **SANDBOX-03**: Runtime apply restartuje terminály, přepíná file tree root a přemapuje otevřené taby
- [x] **SANDBOX-04**: OFF je blokováno při staged souborech; ON nabízí automatický sync do sandboxu

### UI Panely a Widgety

- [x] **UI-01**: Menu bar, dialogy a všechny egui widgety respektují aktivní téma (bez hardcoded tmavých barev)
- [x] **UI-02**: Záložky editoru — indikátor neuloženého stavu (●) je čitelný v obou módech
- [x] **UI-03**: Status bar text je čitelný v obou módech

## v2 Requirements

### Budoucí rozšíření

- **SYS-01**: Automatická detekce OS dark/light preference — experimentální, odložit
- **CUSTOM-01**: Vlastní theme editor — příliš komplexní pro v1
- **SWITCH-01**: Rychlý přepínač tématu v menu Zobrazit — odložit

## Out of Scope

| Feature | Důvod |
|---------|-------|
| OS auto-detect dark/light | Experimentální v egui, zdroj bugů; záměrně vynecháno |
| Vlastní theme editor | Mimo rozsah tohoto milestonu |
| Import/export témat | Nepotřebné pro v1 |
| Animované přechody mezi tématy | Nedostupné v egui |
| Přepínač v menu Zobrazit | Jen Settings panel (dle rozhodnutí) |
