# Akční plán oprav (Audit 2026-02-22) — HOTOVO

Tento plán vychází z auditů `AUDIT-2026-02-22_01.md` a `AUDIT-2026-02-22_02.md`.

## 1. Stabilita a Bezpečnost (Kritické)
- [x] **Mutex Safety:** Nahrazeno `.lock().unwrap()` za `.expect("context")` v celém projektu. Odstraněno riziko tiché paniky bez diagnostiky.
- [x] **Data Integrity (K-1):** Přidán `read_error` flag do `Tab`. Pokud čtení selže, tab je v režimu read-only a nelze jej uložit (brání přepsání souboru chybovou hláškou). Přidány překlady.
- [x] **Terminal Fix:** Ošetřen integer underflow v indexaci mřížky terminálu (`src/app/ui/terminal.rs`).
- [x] **LSP Stability:** Přidán timeout (10s) pro inicializaci LSP klienta v `src/app/lsp/mod.rs`.

## 2. Výkon a Odezva UI (Vysoké)
- [x] **Async I/O:** Synchronní `read_to_string` v `process_background_events` přesunuto do background tasků. Výsledky jsou zpracovávány asynchronně přes `background_io_rx`.
- [x] **Watcher Limiter:** Omezen počet zpracovaných eventů na 500 na jeden frame v `FileWatcher` i `ProjectWatcher`.
- [x] **Sandbox Optimization:** 
    - Sandbox už není ve filtr ignorovaných cest watcheru (umožňuje event-driven refresh).
    - Odstraněn periodický scan (3s). Refresh se spouští pouze při dirty flagu.
    - Optimalizována deduplikace v `get_staged_files` přes `HashSet` (změna z O(n²) na O(n)).
- [x] **Render Loop:** Optimalizováno klonování `Settings` (použit `Arc<Settings>` v `AppShared`) a velkých vektorů souborů v `ProjectIndex` (použit `Arc<Vec<PathBuf>>`). Odstraněno klonování markdown obsahu při renderování preview.

## 3. Optimalizace a Logika (Střední)
- [x] **LSP Throttling:** Implementován debounce (500ms) a `lsp_synced_version` pro `didChange` notifikace. Celý dokument se už neposílá při každém stisku klávesy.
- [x] **Markdown Preview:** Odstraněno klonování obsahu každý frame (předávána reference).
- [x] **Git Logic:** Opraven off-by-one při parsování rename záznamů v git status (zajištěno správné zobrazení cílové cesty).
- [x] **Syntax Highlighting:** Přidána MRU mezipaměť (posledních 20 výsledků) do `Highlighter` pro zamezení full-text re-highlightingu každý frame.

## 4. Ostatní
- [x] **IPC Recovery:** Implementována detekce a obnova `.tmp` souborů v `src/ipc.rs` při startu, pokud hlavní soubor session/recent chybí.
- [x] **UI Feedback:** Přidán textový indikátor "LSP se inicializuje..." do status baru.
